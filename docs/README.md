# Документация

Принцип наименования файлов: **`{тип}-{тема}.md`**

| Префикс   | Назначение                                      | Примеры                    |
| --------- | ------------------------------------------------ | -------------------------- |
| `create-` | Пошаговые инструкции по созданию                 | create-entity, create-form |
| `use-`    | Использование компонента или паттерна            | use-modal                  |
| `guide-`  | Руководства по процессам и практикам             | guide-documentation, guide-ui-and-responsive |
| `config-` | Конфигурация инструментов                        | config-linter              |
| `ref-`    | Справочники (команды, API, опции)                | ref-npm-scripts            |

Формат: kebab-case, расширение `.md`.
